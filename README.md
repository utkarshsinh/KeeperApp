# KeeperApp

This is a Google Keep inspired, Notes keeping app - "Keeper" to manage all your notes. It has been developed using React and JSX and styled with CSS and Material UI.


**Tech Stack-**

* HTML
* CSS
* React Js (JSX)
* Material UI
